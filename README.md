# О проекте

Тестовое задание на позицию junior+ разработчика, найденное в интернете. Задача представляет собой краткую версию сайта вида "интернет-магазин": можно просматривать список астероидов, информацию о каждом из них, а также добавлять их в корзину и подтверждать заказ. Проект реализован на 16-й версии next.js. Полный текст задачи можно найти по ссылке: https://docs.google.com/document/d/1bSC3hgaYe69FJFKKNFHKokJ2Rs0bxkYQ9ixnBS8xn-M/edit?tab=t.0#heading=h.zhvqi4h9zsj2

## Запуск проекта

- npm run dev - запуск проекта для локальной разработки
- npm run build - сборка проекта
- npm run storybook - запуск storybook

## Основные характеристики проекта

- Проект на next 16, в котором используется нововведение cache components
- В качестве прокси между сервером nasa и клиентской частью использованы route handlers, а для обновления состояния корзины - server actions
- Для реализации бесконечного скролла использованы tanstack query и tanstack virtual
- Состояние корзины сохраняется в базе, которая в свою очередь подключена через sequelize (поскольку это тестовый проект, для простоты не исключил сам файл базы данных из репозитория)
- Для визуальных тестов использован storybook
- Стили добавлены через модули css - согласно тексту задачи
